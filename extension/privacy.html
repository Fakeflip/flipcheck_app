1. Verantwortlicher

Verantwortlich im Sinne der DSGVO für die Flipcheck Chrome Extension ist:
Flipcheck · Kontakt: hello@joinflipcheck.app

2. Welche Daten werden verarbeitet?

Die Extension verarbeitet ausschließlich folgende Daten:

a) EAN / ASIN (Produktkennzeichnungen)

Werden vom Nutzer eingegeben oder automatisch von Produktseiten extrahiert, um Marktdaten abzufragen. Diese Daten werden an api.joinflipcheck.app übertragen.

b) Authentifizierungstoken (JWT)

Wird in chrome.storage.local gespeichert. Dient der Authentifizierung gegenüber dem Flipcheck-Backend. Verlässt das Gerät nur als Bearer-Token im HTTP-Header.

c) Nutzereinstellungen

Standard-Kategorie, Modus, Versandkosten u.a. werden lokal in chrome.storage.local gespeichert. Diese Daten verlassen das Gerät nicht.

d) Letzte Checks (Recent)

Die letzten 6 Checks (EAN/ASIN, Verdict, Profit) werden lokal in chrome.storage.local gespeichert. Kein Server-Sync.

3. Welche Daten werden NICHT verarbeitet?

  • Keine Browserhistorie, keine Cookies, keine Passwörter
  • Keine personenbezogenen Daten außer dem optionalen Discord-Profil (Name, E-Mail) — ausschließlich für das Nutzer-Profil angezeigt
  • Kein Tracking, kein Remarketing, keine Werbung
  • Keine Weitergabe an Dritte
  • Keine Seiteninhalt-Analyse außer EAN/ASIN-Extraktion auf Produktseiten

4. Rechtsgrundlage der Verarbeitung

Die Verarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. b DSGVO (Vertragserfüllung / Nutzung des Dienstes) und Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der Bereitstellung der Kernfunktionen).

5. Datenübertragung an Dritte

API-Anfragen werden an api.joinflipcheck.app gesendet. Dieser Server wird von Flipcheck betrieben. Es erfolgt keine Weitergabe an weitere Dritte. Der Server ist in der EU/EWR gehostet.

6. Datenspeicherung und Löschung

Lokal gespeicherte Daten (Token, Einstellungen, Recent) können jederzeit gelöscht werden:

  • Über Einstellungen → Abmelden in der Extension
  • Durch Deinstallation der Extension (löscht alle lokalen Daten)
  • Über die Chrome-Einstellungen: Erweiterungen → Flipcheck → Details → Daten löschen

7. Berechtigungen der Extension

Die Extension benötigt folgende Chrome-Berechtigungen:

  • storage — Für lokale Speicherung von Token und Einstellungen
  • alarms — Für regelmäßige Cache-Bereinigung
  • contextMenus — Für Rechtsklick-Menü (EAN direkt prüfen)
  • tabs — Für die automatische ASIN-Erkennung der aktiven Tab-URL
  • host_permissions auf eBay.de, Amazon.de, Kaufland.de u.a. — Um Badges und Panels auf Produktseiten einzublenden und EANs zu extrahieren
